⚙️ Configurar Firebase
⚠️ O sistema precisa do Firebase configurado para funcionar. Siga os passos abaixo — leva cerca de 10 minutos.
1

Criar conta no Google Firebase

Acesse console.firebase.google.com e entre com sua conta Google.

Clique em "Criar projeto", dê o nome grafica-crm e desative o Google Analytics (não precisa).

2

Ativar o banco de dados

No menu esquerdo clique em Realtime Database → Criar banco de dados.

Escolha a região us-central1 e no modo de segurança selecione "Modo de teste" (pode mudar depois).

3

Pegar as credenciais

Vá em Configurações do projeto ⚙️ → Geral → Seus apps → Adicionar app → Web (</>).

Dê o nome grafica-crm e clique em Registrar app.

Vai aparecer um bloco de código. Copie os valores de apiKey, authDomain, databaseURL, projectId, storageBucket, messagingSenderId, appId.

4

Colar no arquivo index.html

Abra o arquivo index.html com o Bloco de Notas. Procure pelo trecho:

apiKey: "COLE_AQUI_SUA_API_KEY"

Substitua cada COLE_AQUI... pelo valor correspondente que copiou do Firebase.

5

Subir no Vercel (link gratuito)

Acesse vercel.com → crie conta com Google → clique em Add New → Project.

Arraste a pasta grafica-crm para a tela do Vercel. Clique em Deploy.

Em 1 minuto seu link estará pronto: grafica-crm.vercel.app

Compartilhe esse link com toda a equipe — funciona em qualquer computador ou celular!
